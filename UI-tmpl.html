Pattaya.Global.UI.tmpl

前端模板引擎

  1. tmpl(string selector, object settings)

    概述：

    只适用于二级菜单。 用于简单的菜单显示和隐藏。

    参数说明：

    • selector： menu选择器。
    • settings： 配置项，里面对象说明：
      intervalTrigger：int。鼠标移入触发显示事件间隔 (默认：100)
      IntervalHide：int。鼠标移出触发隐藏事件间隔 (默认：200)
      before：function。定义显示之前执行的事件 (默认：undefined)
      after：function。定义显示之前执行的事件 (默认：undefined)
      posChanged：boolean。 打开的子菜单是否随窗口滚动而位置变化。(默认：true)
      top： 当posChanged为true时， 默认的子菜单定位位置。 (默认：0)
      wintop：posChanged为true时，且窗口出现滚动条时， 弹出的子菜单距窗口顶端的距离。 (默认：10)

    Example：

    • 示例1：
      HTML结构：
      <div class="menuTest" style="background:#f60;width:200px;margin:10px 0"> <p>一级菜单</p> <div style="display:none;padding:0 0 0 10px">二级菜单</div> </div> 
      Pattaya.Global.UI.Menu.toggle(".menuTest", {posChanged:false});
