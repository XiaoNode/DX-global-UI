Pattaya.Global.UI.popup

需要引用popup.css，
依赖Pattaya.Global.ns，Pattaya.Global.UI.zIndex方法。

  1. popup(string selector, object settings)

    概述：

    用于popup弹框插件，

    参数说明：

    • selector： popup弹出的内容。
    • settings： popup配置项：
      autoOpen：boolean。调用popup是否自动弹出 (默认：true)
      autoClose：boolean。 点击其他地方，是否自动关闭popup。 (默认：false)
      width：int。设置popup的宽度
      height：int。设置popup的高度
      popupClass：string。自定义popup样式class名

    返回值说明：

    返回值包含4个对象分别如下：
    • show()：显示popup
    • hide()：隐藏popup
    • pos({left:, top:})：定位popup
    • unload()：remove popup

    Example：

    • 示例1：
      window.pp = Pattaya.Global.UI.popup('#test', {
	title: '哈哈',
	width: '432px'
});
