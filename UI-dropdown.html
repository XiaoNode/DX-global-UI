Pattaya.Global.UI.dropdown

——模拟下拉框 UI组件。

dropdown(string selector, object options)

selector：dropdown选择器

options内配置说明：

  • trigger： string。 触发事件类型 (默认："mouseover")
  • type： string ['menu' | 'select']。 设定下拉框点击后不同效果 (默认："select")
  • delay： int。 mouseover/mouseenter触发事件事件间隔 (默认：20[ms])
  • fx： object，下拉效果。对象包括type，值为: 'slideDown'(下拉显示) or 'none'(直接显示)； duration,效果持续时间，默认：200[ms]
  • onChanging： function(curItem)。 点击下拉选项，效果发生前执行的事件 (默认：undefined)。参数为当前点击的option选项(jquery对象)
  • onChanged： function(curItem)。 点击下拉选项，效果发生后执行的事件 (默认：undefined)。参数为当前点击的option选项(jquery对象)

示例

绑定dropdown事件，HTML结构如下：
<style> .dp {width: 100px; margin: 15px 0;} a.hovered {background: #f0f;} </style> <div class="testDropDown dp"> <a href="javascript:;" style="display:block;">点击我</a> <ul style="display:none; background: #ff0;"> <li style="cursor:pointer">点击我</li> <li style="cursor:pointer">点击你</li> <li style="cursor:pointer">点击他</li> </ul> </div> 
JS绑定：
						
Pattaya.Global.UI.dropdown(".testDropDown", {
	trigger: "click",
	onChanged: function(){
		console.log("aaa");
	}
});
						
绑定dropdown事件，HTML结构如下：
<div class="testDropDown2 dp"> <a href="javascript:;" style="display:block;">放在我上面</a> <ul style="display:none; background: #ff0;"> <li style="cursor:pointer">放在我上面</li> <li style="cursor:pointer">放在你上面</li> <li style="cursor:pointer">放在他上面</li> </ul> </div> 
JS绑定：
						
Pattaya.Global.UI.dropdown(".testDropDown2", {
	type: 'menu',
	onChanged: function(){
		console.log("bbb");
	}
});
