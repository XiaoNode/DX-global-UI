Pattaya.Global.UI.numerical(string filter, object options)

数量操作的ui组件。目前主要用于产品数量的输入。
需要引用jQuery.caret.js
filter为输入框选择器， options为一系列配置参数。

options内配置说明：

  • step： 实数。 每一次递增的数值 (默认：1)
  • decCtrl： string。 递减控制器 (默认：undefined)
  • incCtrl： string。 递增控制器 (默认：undefined)
  • max： int。 输入框内所能输入的最大数值 (默认：undefined)
  • min： int。 输入框内所允许输入的最小数值 (默认：undefined)
  • defaultValue： int。 输入框的默认值 (默认：0)
  • decimal： boolen。 输入框值显示为float还是int？ (默认：false)
  • decimalDelimiter： string。 设置小数点的显示符号 (默认：".")
  • decimalPrecision： string。 设置小数点的显示位数 (默认：2[保留2位小数点])
  • onValueChanging： function。 输入框值改变之前执行的事件 (默认：undefined)
  • onValueChanged： function。 输入框值改变之后执行的事件 (默认：undefined)
  • styles： object。 输入框的样式变化[暂未启用改参数设置] (默认：{error: 'error'}，)

返回值为obj类型，下其包含6个对象，各对象说明如下：

  • increase()： function。 执行加法事件
  • decrease()： function。 执行减法事件
  • unload()： function。 初始化之前执行的事件
  • currentValue()： function。 获取当前输入框值
  • enable()： function。 启用numerical
  • disable()： function。 禁用numerical

Example：

HTML：
<div class="exe"> <a class="dec" href="javascript:;">-</a><input class="testNumerical" type="text" /><a class="inc" href="javascript:;">+</a> </div> 
Javascript：
					
window.numerical = Pattaya.Global.UI.numerical(".testNumerical", {
	decCtrl: ".dec",
	incCtrl: ".inc",
	max: 30,
	min: 2,
	defaultValue: 2,
	onValueChanged: function(nv, ov) {
		console.log(nv+ ' ' + ov);
	}
});
					
numerical.increase(); //执行increase()
					
numerical.decrease(); //执行decrease()
					
alert(numerical.currentValue()); //输出currentValue
					
alert(numerical.currentValue()); //输出currentValue
					
numerical.enable(); //启用
					
numerical.disable(); //禁用
