Pattaya.Global.UI.tabs(string selector, object options)

tab切换。
filter为tab选择器， options为一系列配置参数。

options内配置说明：

  • active： integer。 激活项的索引 (默认：-1, 表示不管)，从0还是计数
  • trigger： string。 触发事件类型 (默认："click")
  • triggerInterval： int。 mouseover/mouseenter触发事件事件间隔 (默认：150[ms])
  • onChanging： function(pre, cur)。 切换前执行的事件，参数分为上一个触发器和当前触发器 (默认：undefined)
  • onChanged： function(pre, cur)。 切换后执行的事件，参数分为上一个触发器和当前触发器 (默认：undefined)

示例

绑定tab事件，HTML结构如下：
<style> .tabTest .active {background: #ff0;} .conTest div {display: none;} .conTest .active {display:block;} </style> <ul class="tabTest clearfix"> <li class="active fl"><a href="#wo">tab1</a></li> <li class="fl"><a href="#ni">tab2</a></li> </ul> <div class="conTest"> <div id="wo" class="active">我</div> <div id="ni" >你</div> </div> 
JS绑定：
					
Pattaya.Global.UI.tabs(".tabTest", {
	trigger: "mouseover",
	triggerInterval: 200,
	onChanged: function(){
		console.log("aaa");
	}
});
